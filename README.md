# RAAJRANI POS — Point of Sale for 200+ Bangladeshi Businesses

Cloud POS built for retail shops in Bangladesh. Handles inventory, sales, bKash/Nagad, and low-internet environments.

## 🔥 Real Impact

- **200+ active stores** (Bogura, Dhaka, Rajshahi)
- **13 years** in production
- Works offline, syncs when online

## ⚙️ Core Features

- Inventory management with barcode
- Sales & returns
- bKash, Nagad, COD integration
- Multi-store dashboard
- Daily profit/loss reports
- Low-bandwidth optimized

## 🛠 Built With

- PHP 7/8, MySQL
- JavaScript, jQuery
- REST API for mobile app
- Redis for session caching

## 🎯 Why It Works in BD

Designed for shops with 2G internet, power cuts, and non-technical staff. One-click training included.
