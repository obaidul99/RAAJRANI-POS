# RAAJRANI POS — ERP for Bangladeshi SMEs

Not a simple POS. A 16-module production ERP running 200+ businesses across Bangladesh for 13 years.

## 🧩 Modules

- **Dashboard** — Real-time sales, profit, stock alerts
- **Sales Manager** — POS, wholesale, returns, credit sales
- **Product Manager** — Variants, barcode, expiry tracking
- **Purchase Manager** — Supplier ledger, GRN
- **Accounts Manager** — Double-entry, P&L, balance sheet
- **Production Manager** — BOM, raw material consumption, finished goods
- **Inventory & Warehouse** — Multi-location stock
- **HR Manager** — Attendance, payroll, shifts
- **Asset Manager** — Depreciation tracking
- **Vehicle Manager** — Fuel, maintenance, trips
- **Agro Manager** — Crop cycles, farm expenses (unique to BD)
- **Zakat Manager** — Islamic finance calculation
- **SMS Manager** — Bulk SMS for due collection
- **Report Manager** — 50+ reports, export to Excel
- **Admin Area** — Role-based permissions
- **Settings & Support**

## 🚀 Scale

- 200+ live installations
- Handles 10,000+ products per store
- Works offline, syncs via API
- Built for 2G networks and power cuts

## 🛠 Tech

PHP 8, MySQL (complex joins), JavaScript, REST API, Redis, Cron jobs for daily closing
